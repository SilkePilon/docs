### Real-time Updates WebSocket
Connect to our community-maintained WebSocket endpoint to receive instant updates and participate in article discussions.

**WebSocket URL:**
<code>wss://api.jotihunt.live/ws</code>

**Features:**
- Real-time game updates (hints, assignments, articles)
- Article discussion chat system
- No authentication required
- Automatic reconnection
- Low latency delivery
- Unlimited connections
</Check>

<Note>
### Game Updates Channel
Here's an example of connecting to the game updates channel:

```javascript
const ws = new WebSocket('wss://api.jotihunt.live/ws');

ws.onopen = () => {
  console.log('Connected to Jotihunt live updates');
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  
  switch (data.type) {
    case 'hint':
      console.log('New hint for area:', data.data.area);
      console.log('Hint content:', data.data.content);
      break;
      
    case 'assignment':
      console.log('New assignment:', data.data.title);
      console.log('Points:', data.data.points);
      console.log('Description:', data.data.description);
      break;
      
    case 'article':
      console.log('New article:', data.data.title);
      console.log('Category:', data.data.category);
      break;
  }
};

// Error handling
ws.onerror = (error) => {
  console.error('WebSocket error:', error);
};

ws.onclose = () => {
  console.log('Connection closed, attempting to reconnect...');
  setTimeout(() => {
    // Reconnect logic
  }, 5000);
};
```
</Note>

<Note>
### Article Chat Channel
Example of using the article chat system:

```javascript
const chatWs = new WebSocket('wss://api.jotihunt.live/ws');

// Listen for chat messages
chatWs.onmessage = (event) => {
  const data = JSON.parse(event.data);
  
  if (data.type === 'chat') {
    console.log(`${data.data.username}: ${data.data.message}`);
    
    if (data.data.replyTo) {
      console.log('Replying to earlier message');
    }
  }
};

// Send a chat message
function sendChatMessage(articleId, message, replyToId = null) {
  const chatMessage = {
    type: 'chat',
    data: {
      articleId: articleId,
      message: message,
      userId: 'your-user-id', // Replace with actual user ID
      username: 'your-username', // Replace with actual username
      replyTo: replyToId
    },
    timestamp: new Date().toISOString()
  };
  
  chatWs.send(JSON.stringify(chatMessage));
}
```
</Note>

<Warning>
### Implementation Guidelines
- Implement proper error handling and reconnection logic
- Handle all message types appropriately
- Validate message format before sending
- Keep track of connection state
- Consider implementing an exponential backoff for reconnection attempts
</Warning>

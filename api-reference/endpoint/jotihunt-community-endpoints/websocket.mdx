### Real-time Updates WebSocket
Connect to our community-maintained WebSocket endpoint to receive instant updates without polling the REST API.

**WebSocket URL:**
<code>wss://api.jotihunt.live/ws</code>

**Features:**
- Real-time data updates
- No authentication required
- Automatic reconnection
- Low latency delivery
- Unlimited connections
</Check>

<Note>
### Usage Example
Here's a simple example of connecting to the WebSocket:

```javascript
const ws = new WebSocket('wss://api.jotihunt.live/ws');

ws.onopen = () => {
  console.log('Connected to Jotihunt live updates');
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Received update:', data);
};
```

The WebSocket server runs on dedicated infrastructure to ensure 24/7 availability and minimal latency.
</Note>

<Warning>
While the WebSocket connection is free to use, please implement appropriate error handling and reconnection logic in your applications to ensure smooth operation.
</Warning>
